Интерактивные уроки по Python для новичков

Здесь ты найдёшь 50 простых задач с объяснениями. Пиши код, проверяй себя и учись шаг за шагом!

Задача 1: Привет, мир!

Давай начнём с самого простого. В Python есть команда print(), которая выводит текст на экран. Представь, что ты здороваешься с компьютером. Напиши код, чтобы вывести "Привет, мир!". Просто введи это в скобках, и всё сработает.


Задача 2: Имя на экране

Теперь давай выведем твоё имя. Напиши код, который выведет "Меня зовут [твоё имя]". Например, если тебя зовут Саша, напиши print("Меня зовут Саша"). Текст должен быть в кавычках!


Задача 3: Любимое число

Числа в Python писать легко — без кавычек. Напиши код, чтобы вывести своё любимое число. Например, если это 7, просто напиши print(7). Попробуй!


Задача 4: Переменная с именем

Переменные — это как коробки, куда можно положить данные. Создай переменную name и присвой ей своё имя (в кавычках). Потом выведи её с помощью print(). Например: name = "Саша", затем print(name).


Задача 5: Число в переменной

Теперь попробуем с числом. Создай переменную age и присвой ей свой возраст. Выведи её на экран. Например: age = 15, затем print(age). Без кавычек для чисел!


Задача 6: Приветствие с переменной

Давай соединим текст и переменную. Создай переменную name = "Твоё имя" и выведи "Привет, [имя]!" с помощью print(). Используй плюс: print("Привет, " + name).


Задача 7: Сколько тебе лет?

Создай переменную age с твоим возрастом и выведи "Мне [возраст] лет". Например: age = 15, затем print("Мне " + str(age) + " лет"). str() превращает число в текст.


Задача 8: Сложение чисел

Python умеет считать! Создай две переменные a = 5 и b = 3, сложи их и выведи результат. Напиши: print(a + b). Должно получиться 8.


Задача 9: Вычитание

Теперь вычтем. Создай x = 10 и y = 4, выведи разницу с помощью print(x - y). Должно быть 6.


Задача 10: Умножение

Умножение — это звёздочка *. Создай a = 6 и b = 2, выведи их произведение: print(a * b). Должно быть 12.


Задача 11: Деление

Деление — это слеш /. Создай a = 15 и b = 3, выведи результат деления: print(a / b). Должно быть 5.


Задача 12: Больше или меньше?

Проверим условие. Создай a = 7 и выведи "Да", если оно больше 5. Используй if: if a > 5: и print("Да"). Не забудь отступ!


Задача 13: Равенство

Равенство проверяется двумя знаками ==. Создай x = 10 и выведи "Равно", если оно равно 10. Пиши: if x == 10: и print("Равно").


Задача 14: Не равно

Неравенство — это !=. Создай y = 8 и выведи "Не равно", если оно не равно 5. Пиши: if y != 5: и print("Не равно").


Задача 15: Меньше или равно

Проверка "меньше или равно" — это <=. Создай z = 4 и выведи "ОК", если оно меньше или равно 5. Пиши: if z <= 5: и print("ОК").


Задача 16: Больше или равно

Проверка "больше или равно" — это >=. Создай n = 6 и выведи "Да", если оно больше или равно 6. Пиши: if n >= 6: и print("Да").


Задача 17: Если-иначе

Добавим else — это "иначе". Создай a = 3. Если оно больше 5, выведи "Больше", иначе "Меньше". Пиши: if a > 5: и else:.


Задача 18: Проверка возраста

Создай age = 18. Если возраст больше или равен 18, выведи "Взрослый", иначе "Ребёнок". Используй if и else.


Задача 19: Чётное число

Чётное число делится на 2 без остатка — используй %. Создай n = 4. Если n % 2 == 0, выведи "Чётное".


Задача 20: Нечётное число

Теперь проверим нечётное. Создай n = 7. Если n % 2 != 0, выведи "Нечётное".


Задача 21: Простой цикл

Цикл for повторяет действия. Выведи числа от 1 до 3 с помощью for i in range(1, 4): и print(i).


Задача 22: Привет 5 раз

Повтори "Привет" 5 раз. Используй for i in range(5): и print("Привет"). range(5) — это от 0 до 4.


Задача 23: Сумма чисел

Создай переменную sum = 0. Сложи числа от 1 до 5 в цикле и выведи сумму. Используй for i in range(1, 6): и sum = sum + i.


Задача 24: Таблица умножения

Выведи таблицу умножения на 2 (2, 4, 6, 8, 10). Используй for i in range(1, 6): и print(i * 2).


Задача 25: Пока меньше 10

Цикл while работает, пока условие верно. Создай n = 0 и выведи числа до 10 с шагом 1: while n < 10:, print(n), n = n + 1.


Задача 26: Обратный отсчёт

Создай n = 5 и выведи числа от 5 до 1. Используй while n > 0:, print(n), n = n - 1.


Задача 27: Список друзей

Список — это как коробка с вещами. Создай friends = ["Аня", "Коля", "Маша"] и выведи первый элемент: print(friends[0]). Нумерация с 0!


Задача 28: Все друзья

Выведи всех друзей из списка friends = ["Аня", "Коля", "Маша"] в цикле: for friend in friends: и print(friend).


Задача 29: Длина списка

Функция len() считает элементы. Создай nums = [1, 2, 3, 4] и выведи длину: print(len(nums)). Должно быть 4.


Задача 30: Добавление в список

Метод append() добавляет элемент. Создай numbers = [1, 2], добавь 3 и выведи: numbers.append(3), print(numbers).


Задача 31: Удаление из списка

Метод pop() убирает элемент. Создай items = ["яблоко", "банан"], удали последний и выведи: items.pop(), print(items).


Задача 32: Максимум в списке

Функция max() ищет максимум. Создай nums = [4, 2, 8, 1] и выведи максимум: print(max(nums)). Должно быть 8.


Задача 33: Минимум в списке

Функция min() ищет минимум. Создай nums = [4, 2, 8, 1] и выведи минимум: print(min(nums)). Должно быть 1.


Задача 34: Простая функция

Функции — это как рецепты. Создай функцию say_hello(), которая выводит "Привет". Пиши: def say_hello(): и print("Привет"), затем вызови say_hello().


Задача 35: Функция с аргументом

Добавим аргумент. Создай функцию greet(name), которая выводит "Привет, [имя]". Пиши: def greet(name): и print("Привет, " + name), затем greet("Саша").


Задача 36: Функция с числом

Создай функцию double(n), которая умножает число на 2 и выводит. Пиши: def double(n): и print(n * 2), затем double(5).


Задача 37: Возврат значения

Функция может возвращать результат с return. Создай def add(a, b):, верни сумму и выведи: return a + b, затем print(add(3, 4)).


Задача 38: Проверка чётности

Создай функцию is_even(n). Если число чётное (n % 2 == 0), верни "Чётное", иначе "Нечётное". Выведи: print(is_even(6)).


Задача 39: Сравнение чисел

Создай функцию compare(a, b). Если a > b, выведи "Первое больше", иначе "Второе больше или равно". Выведи: compare(5, 3).


Задача 40: Простое число

Создай функцию is_prime(n). Если n делится только на 1 и себя, выведи "Простое". Для n = 7 проверь с циклом до 6.


Задача 41: Факториал

Факториал — это произведение чисел от 1 до n. Создай функцию factorial(n) для n = 4. Используй цикл и return.


Задача 42: Реверс строки

Создай переменную s = "Python" и выведи её задом наперёд с помощью print(s[::-1]). Должно быть "nohtyP".


Задача 43: Длина строки

Создай s = "Привет" и выведи длину с print(len(s)). Учитываются все символы, включая пробелы.


Задача 44: Заглавные буквы

Создай s = "python" и выведи с заглавными буквами: print(s.upper()). Должно быть "PYTHON".


Задача 45: Маленькие буквы

Создай s = "HELLO" и выведи с маленькими буквами: print(s.lower()). Должно быть "hello".


Задача 46: Замена текста

Создай s = "Я люблю Python" и замени "Python" на "код". Используй print(s.replace("Python", "код")).


Задача 47: Подсчёт символов

Создай s = "абракадабра" и выведи, сколько раз встречается "а": print(s.count("а")).


Задача 48: Срез строки

Создай s = "Python" и выведи первые 3 символа: print(s[0:3]). Должно быть "Pyt".


Задача 49: Последние символы

Создай s = "Программирование" и выведи последние 4 символа: print(s[-4:]). Должно быть "ние".


Задача 50: Поздравление

Создай функцию congratulate(name), которая выводит "С днём рождения, [имя]!". Вызови её с congratulate("Вася").
